Problem Statement 1:
-------------------------
Add a Product to the Shopping Cart
You must design a Java program that manages a shopping cart using an ArrayList<String>.
Your task is to add a new product to an existing cart and display the updated list.

Constraints
The cart stores only Strings.
New product must be added at the end of the list.

Input Format
Create the initial cart.
Add the product “Pencil”.
Print the updated list.

Output Format
Display the cart after insertion.

Sample Input
Initial cart: [Pen, Notebook]
Add: Pencil

Sample Output
Updated cart: [Pen, Notebook, Pencil]

Test Cases
Test Case 1
Input:
Cart = ["Mouse", "Keyboard"]
Add = "USB Cable"

Output:
[Mouse, Keyboard, USB Cable]

Test Case 2
Input:
Cart = []
Add = "Water Bottle"

Output:
[Water Bottle]

Class Design
Class Name: ShoppingCartManager

Variables
private ArrayList<String> cart;

Methods
public void addItem(String item);
public void displayCart();


Problem Statement 2:
-------------------------
Remove an Item from Favorites List
You must design a Java program that removes a specified favorite item from
a user’s list of favorites.

Constraints
Only Strings may be stored.
Removal must use remove(Object o).
If item is NOT in list, no error should occur.

Input Format
Create favorites list.
Remove "Movie2".
Display final list.
Output Format
Display updated favorites list.

Sample Input
Initial favorites: [Movie1, Movie2, Movie3]
Remove: Movie2

Sample Output
Updated favorites: [Movie1, Movie3]

Test Cases
Test Case 1
Input:
List = ["Inception", "Avatar", "Dune"]
Remove = "Avatar"

Output:
[Inception, Dune]

Test Case 2
Input:
List = ["Book1"]
Remove = "Book2"

Output:
[Book1]
(No removal since “Book2” not found.)

Class Design
Class Name: FavoritesManager

Variables
private ArrayList<String> favorites;

Methods
public void removeItem(String item);
public void displayFavorites();

Problem Statement 5:
-----------------------
Validating a Password Reset Request
You must design a Java method that validates a password reset request.
The rule is simple:
- If the old password contains the unlock key (String key), then return the
 modified old password with all occurrences of the key removed.

- If the old password does not contain the key, return the original password unchanged.
This simulates a real-world case where security tokens embedded in passwords must
be stripped before processing.

Constraints
-----------
- Case-sensitive comparison.
- Remove all occurrences of the key, not just the first.
- Input strings may be empty but not null.

Input Format
-----------
The method receives two strings:
oldPassword
unlockKey

Output Format
-----------
Return the processed password string.
Sample Input
oldPassword = "mypasswordtoken12"
unlockKey = "token"

Sample Output
mypassword12

Test Cases
-----------
Test Case 1
Input:
oldPassword = "resetcodecode2025"
unlockKey = "code"

Output:
reset2025

Test Case 2
Input:
oldPassword = "admin123"
unlockKey = "xyz"


Output:
admin123
Class Design
Class Name: PasswordResetValidator

Methods
public String removeKey(String oldPassword, String unlockKey);
