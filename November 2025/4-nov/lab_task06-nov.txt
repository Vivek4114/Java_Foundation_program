Question 1: User Age Validation (Integer Wrapper)
----------------------------------------------------------
Problem Statement:
Write a program to validate a userÆs age entered as a string. You must convert the string to an integer using a wrapper class.
If the conversion fails (invalid input), print "Invalid age". If the conversion succeeds, check if the user is eligible (= 18 years).

Class Name: AgeValidator

Instance Variables:
private String ageInput;

Constructor:
public AgeValidator(String ageInput)

Methods:
public Integer convertToAge() Ś Converts string to Integer, prints "Invalid age" if invalid.
public void checkEligibility() Ś Prints "Eligible" if age = 18 else "Not Eligible".

Input Format:
<age as string>

Output Format:
Eligible / Not Eligible / Invalid age

Sample Test Cases:
Input: 25 ? Eligible
Input: 16 ? Not Eligible
Input: abc ? Invalid age
Input: -5 ? Not Eligible
Input: (empty) ? Invalid age
